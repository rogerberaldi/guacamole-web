Apache Guacamole Common JS Library
====================================

This directory should contain the guacamole-common-js library files.

REQUIRED FILES:
- guacamole-common.min.js (main library file)
- guacamole-common.js (unminified version)
- LICENSE (Apache 2.0 license)

DOWNLOAD INSTRUCTIONS:

1. Download the library:
   wget https://downloads.apache.org/guacamole/1.6.0/binary/guacamole-common-js-1.6.0.tar.gz

2. Extract to this directory:
   tar -xzf guacamole-common-js-1.6.0.tar.gz --strip-components=1

3. Remove archive:
   rm guacamole-common-js-1.6.0.tar.gz

IMPORTANT:
The guacamole-common-js library is NOT included in this repository due to licensing.
You must download it separately from the Apache Guacamole project.

For more information:
https://guacamole.apache.org/
